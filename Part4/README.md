## 育成方針など
### 育成
- 全滅マップばかりなので基本的にパラ吟味は面倒です
- 良成長したらラッキーくらいで
- もっとも, サナキやアイクには必須パラがあるため (詳細不明), 速さ攻撃力あたりを重視してください
- サナキやエリンシアはレベルのわりに初期値が中途半端なのでレベルアップを少し意識しておいたほうがいいかもしれません
- 他はひたすらレベルアップでいい, 大事なのは力と速さ

### アイテム
- 強化手槍, 強化手斧, 遠距離魔法, レスキューが重要です
- 私のプレイではブーツを使わなかったので, 使いどころがなければ, ラフィエル, ハールあたりに使いましょう
- マスタークラウンを輸送隊に送るために持ち物を埋めておくこと
- サナキのために軽い武器がいります
  - ミカヤのお古のライトでも十分なはず

### 終章のメンバー
- 固定
  - アイク: ハンマーで漆黒殺し
  - ミカヤ: 魔力40は便利
  - サナキ: レクスフレイムでオーラに追撃できる
  - クルトナーガ・イナ・サザ: 体当たり
- 任意
  - カイネギス: オーラ殺し
  - ティバーン: 高い攻撃力
  - エリンシア: 対オーラではベオク最強クラス
  - ジル: 戦績2位, ウルヴァン, オーラに追撃できるはず
  - ハール: 戦績1位, 強化銀の斧
  - マーシャ: 飛行, 炎支援が便利, オーラに追撃できる
  - シノン: 本体はバルフレチェ
  - セネリオ: 魔力高い
  - ワユ, ソーンバルケ: 速い
  - ラフィエル: 一択, 常時4人再行動
