## 育成方針など
### 育成

- アイク
  - MDFとMGCを除いて３つ上がればよい、MDFは上げてはいけない
  - 計算上アイクのMDF初期値ならHP最大値でも, 13章でローラのパージ (+レオの支援) でちょうど1ターンキルできる
  - https://www.nicovideo.jp/watch/sm4387530
- セネリオ
  - 1章攻略のキーになるため, MGC, SPDを拠点で上げておく
  - 2章の時点でHP+DEF＞39必須
- ティアマト
  - 初期ステと上限の関係から、HPとSPDおよび他２つ以上が狙い
- シノン: 拠点成長を推奨、STRを毎回あげる
- オスカー: HP、STR、SPD、DEFのうち少なくとも3つ+a
- ボーレ: 魔力を除き３つ上がればよい、SPD推奨
- ワユ: 魔力を除き３つ上がればよい、STRとDEF推奨
- 一部のキャラを除き、拠点成長以上が目標です
  - それほどこだわらなくてもいいはず

### アイテム
- 練成手槍は7章、練成手斧は8章から使用可能です
  - 終章で少なくとも3つずつつくっておく
- 最初のマスクラをハールが使います
- ほりだし物で, トーチ, 竜の盾, シノンの弓, 秘伝の書, 連続の書はほしい。

### スキル
- 俊足, すり抜け, 護り手, 練達, 連続, 大器晩成, 治癒あたりが役立ちます。

### 支援
- 部隊分けを意識した支援の組み合わせを
- ボーレ+ミストで精霊の粉が手に入る
- アイクの支援は13章の邪魔にならない程度に
