## 育成方針など

### 育成
- ミカヤは毎章レベルアップでSTR以外すべて欲しい　HPとSPDは必須か
  - 天使の衣と竜の盾を使用
  - 9章でほとんど上限値
- ジルは初期ステが少し物足りないので極力よい成長をさせる
  - 中盤以降の主力
  - 拠点でHP,STR,SPDが上がるよう、とても大変だが吟味すると後が楽
  - 6章Area2で場合によってはクラスチェンジ
- ノイスは終章の前後でCCすれば十分。攻撃面を重視する
  - 2章3ターン目にSPD13必須、7章でSTR17推奨
- ローラは３部13章の１ターンクリアのため、その時までにMGC25かつ光B及びレオとの支援Aが必要
  - パージのため高いSPD, SKL, 命中推奨
  - レオとの支援の代わりに最上級クラスチェンジでも可
- ブラッドはマップと拠点での成長により3部までにLV20→CCすればよい
  - 7章でSTR13推奨
- ツイハはエリートをつけて経験値を稼ぐべき、余裕があれば吟味

### アイテム
- マスプルと武術書は必須ではないが、あるとよい
- 2章のセイニーは必須
- 俊足・すり抜け・護り手・強化サンダーは傭兵団へ、他は不明
