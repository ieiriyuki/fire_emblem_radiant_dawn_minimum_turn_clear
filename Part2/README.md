## 育成方針など

### 育成
- 実際、育成というほどLVUP機会はない
- 物理クラスは、速さ≧力≧ＨＰ≧守備≧技を意識して最低でも上3つは上げたい
- 魔法クラスは魔力、速さ＋αで
- 2部以降ずっとこんな成長をねらう
- 成長を気にしておくべきユニットはエリンシア・マーシャ・ハール
  - マカロフあとネフェニーかウハラダのどちらか
